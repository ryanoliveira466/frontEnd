#### âœ… `flex-basis`:

* Sets the **starting point** for sizing a flex item **before** growing or shrinking.
* Works **with** `flex-grow` and `flex-shrink`.
* Often **overrides** `width` in Flexbox.

```css

  flex: 1 1 200px; /* Start at 200px, but can grow/shrink */
}
```
